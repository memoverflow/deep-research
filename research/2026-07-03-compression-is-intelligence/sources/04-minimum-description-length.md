MDL: model selection principle where the shortest description of the data is the best model. Origins in information theory (Rissanen 1978), related to Bayesian Information Criterion. Idealized version relates to Kolmogorov complexity and Solomonoff's theory of inductive inference — best model represented by shortest self-extracting archive. MDL principle requires only that shortest description, when executed, reproduce original data without error (two-part codes: program + literal data).
